# Digital Stopwatch

A simple, visually appealing digital stopwatch built with HTML, CSS, and JavaScript. This stopwatch features start, stop, and reset functionalities with a modern black-and-yellow theme.

## Features

- Displays hours, minutes, seconds, and milliseconds.
- Start, stop, and reset buttons with interactive states.
- Responsive design centered on the screen.
- Smooth button animations and hover effects.
- Real-time time tracking using JavaScript.

## How to Use

1. Open `index.html` in a browser.
2. Click the **Play** button to start the stopwatch.
3. Click the **Pause** button to stop the timer.
4. Click the **Reset** button to reset the timer to `00:00:00:00`.
